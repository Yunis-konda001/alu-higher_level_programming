Test cases for BaseGeometry class

1. Valid integers:
   bg.integer_validator("my_int", 12)
   bg.integer_validator("width", 89)

2. Invalid integer - string:
   try:
       bg.integer_validator("name", "John")
   except Exception as e:
       print("[{}] {}".format(e.__class__.__name__, e))
   Expected Output: [TypeError] name must be an integer

3. Invalid integer - zero:

       bg.integer_validator("age", 0)

       print("[{}] {}".format(e.__class__.__name__, e))
   Expected Output: [ValueError] age must be greater than 0

4. Invalid integer - negative:

       bg.integer_validator("distance", -4)
